# 📧 Email Guardian AI – Spam Detection System

Email Guardian AI is an intelligent email analysis tool that detects whether an email is **Spam or Ham (Legitimate)** using AI-powered analysis.
The system scans email content, sender details, and subject lines to identify phishing or malicious emails.

This project is built with **React + TypeScript** and integrates AI analysis using **Google Gemini AI**.

---

# 🚀 Features

* 🔍 AI-powered **Spam vs Ham detection**
* 📧 Analyze **email sender, subject, and body**
* 🧠 Uses **Gemini AI** for intelligent classification
* 📊 **Analytics dashboard** for email scan history
* 🗂 **History tracking** of analyzed emails
* ⚡ Fast and responsive **React UI**
* 🔐 Privacy-friendly (data stored locally)

---

# 🧠 How It Works

1. User enters:

   * Sender email
   * Subject
   * Email body

2. The application sends the email data to the AI service.

3. The AI analyzes the email and classifies it as:

   * **SPAM** 🚨
   * **HAM (Safe Email)** ✅

4. Results are displayed with analysis details.

# 📊 Example Email Inputs

### Spam Example

Sender:

```
promo@amazon-gift.co
```

Subject:

```
You won $500 gift card
```

Body:

```
Click the link to claim your prize now!
```

Result → **SPAM**

---

### Legitimate Email Example

Sender:

```
john@company.com
```

Subject:

```
Project update
```

Body:

```
Please review the Q3 roadmap and provide feedback.
```

Result → **HAM**
